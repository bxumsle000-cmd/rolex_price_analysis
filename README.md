# 🕐 Rolex 手錶價格分析系統

一個幫助你分析勞力士手錶二手市場價格的 Python 工具,讓你在購買前能夠評估價格是否合理。

## 📋 功能特色

- ✅ 市場價格統計分析(平均價、中位數、價格區間)
- ✅ 賣家報價評估與評分系統
- ✅ 條件、配件、年份等因素的價格影響分析
- ✅ 尋找最相似的交易記錄
- ✅ 異常值偵測(識別不合理的價格)
- ✅ 保值率分析與未來價格預測
- ✅ 視覺化圖表展示

## 📊 資料檔案格式

`data/data_clean.csv` 應該包含以下欄位:

| 欄位名稱 | 說明 | 範例 |
|---------|------|------|
| reference number | 錶款型號 | 116610LN |
| price | 價格(美金) | 12500 |
| condition | 手錶狀況 | Very good |
| age | 錶齡(年) | 5 |
| full_set | 是否全配(1/0) | 1 |
| has_box | 有錶盒(1/0) | 1 |
| has_papers | 有證書(1/0) | 1 |

## 🚀 使用方法

### 1. 執行程式

```bash
python _05_price_analysis.py
```

### 2. 輸入查詢資訊

程式會詢問你三個問題:

```
請輸入 Reference Number (例如: 116610LN):
請輸入賣家報價 (USD):
請輸入手錶年份:
```

**輸入說明:**
- **Reference Number**: 手錶型號,通常在錶盒或證書上
- **賣家報價**: 賣家開出的價格(美金)
- **手錶年分**: 手錶製造年份(例如: 2018)

### 3. 查看分析結果

程式會依序執行以下 10 個分析步驟:

#### Step 1-3: 資料載入與篩選
- 載入市場資料
- 篩選相同型號的手錶
- 顯示找到多少筆資料

#### Step 4: 價格統計分析
會顯示該型號的:
- 平均價格、中位數價格
- 標準差(價格波動程度)
- 最低價、最高價
- 第一四分位數(Q1)、第三四分位數(Q3)

#### Step 5: 賣家價格評估
會告訴你:
- 賣家報價在市場上的百分位排名
- 與平均價、中位數的差距
- **評級與評分**(30-90分)
- **購買建議**

#### Step 6: 條件細分分析
分析以下因素對價格的影響:
- 手錶狀況(New, Unworn, Very good, Good 等)
- 配件完整度(Full Set vs Not Full Set)
- 不同年份的價格趨勢

#### Step 7: 最相似交易記錄
顯示 5 筆最接近你查詢條件的交易:
- 價格、條件、年份
- 所在地區
- 配件情況

#### Step 8: 異常值分析
使用統計方法檢測:
- 計算合理價格範圍
- 判斷賣家報價是否異常
- 給出相應的警告或確認

#### Step 9: 保值率分析
如果資料充足,會顯示:
- 統計模型的可信度
- 每年升值/貶值幅度
- 5年後價格預測
- 保值率百分比

#### Step 10: 視覺化圖表
自動產生 4 個分析圖表:
1. **價格分布直方圖** - 顯示價格分布與賣家報價位置
2. **價格箱型圖** - 顯示價格區間與異常值
3. **各條件平均價格** - 比較不同條件的價格差異
4. **價格 vs 年份趨勢** - 顯示保值率變化

## 💡 評分系統說明

| 評分 | 評級 | 說明 |
|------|------|------|
| 90分 | 價格較低 | 低於市場 25%,可能是好機會 |
| 70分 | 價格偏低 | 低於中位數,相對合理 |
| 50分 | 市場中上 | 略高於平均,常見範圍 |
| 30分 | 價格較高 | 高於市場 75%,需謹慎考慮 |

## ⚠️ 注意事項

### 購買前檢查清單

1. ✅ 確認手錶真偽(要求購買證明)
2. ✅ 檢查實際狀況是否符合描述
3. ✅ 確認保固和售後服務
4. ✅ 要求更多實物照片
5. ✅ 考慮使用第三方驗證服務

### 警告訊息說明

- **⚠️ 價格低於正常範圍**: 可能是絕佳機會,但也可能有問題,需特別注意真偽
- **⚠️ 價格高於正常範圍**: 建議謹慎考慮,尋找其他選項
- **⚠️ 趨勢不顯著**: 保值率分析僅供參考,價格波動較大

## 🔍 常見問題

### Q1: 找不到我的 Reference Number?
**A:** 檢查輸入格式是否正確,程式會顯示資料庫中最常見的型號供參考。

### Q2: 資料從哪裡來?
**A:** 知名二手手錶網站[Chrono24](https://www.chrono24.tw/)，資料來源為kaggle網站資料([Rolex on Chrono24](https://www.kaggle.com/datasets/vittoriohaardt/rolex-on-chrono24))

### Q3: 評分是否絕對準確?
**A:** 評分僅供參考,還需考慮手錶狀況、配件完整度、賣家信譽等因素。

### Q4: 保值率預測可信嗎?
**A:** 預測基於歷史資料,未來市場可能變化,僅作參考用途。
